嚔る

嚔る（はなひる）

くしゃみをする。

>「縁側でひとり—る休日」

出典：『漢検 漢字辞典（第二版）』

粥ぐ

+ PUSH

粥ぐ（ひさぐ）

物を売って商売をする。「駄菓子やおもちゃを—ぐ店」

出典：『漢検 漢字辞典（第二版）』

挈げる

+ PUSH

挈げる（ひっさげる）

手にひっかけて持つ。「かばんを—げて歩く」

出典：『漢検 漢字辞典（第二版）』

繙く

+ PUSH

繙く（ひもとく）（紐解く）

書物をめくり返して読む。「絵巻物を—く」「注釈書を—く」

出典：『漢検 漢字辞典（第二版）』

恢い

+ PUSH

恢い（ひろい）

中があいていて大きい。志が大きい。また、心がひろい。

>「彼女は—い人物だ」

出典：『漢検 漢字辞典（第二版）』

篩

+ PUSH

篩（ふるい）

枠の底に網を張り、振り動かして、粉や粒を粗いものと細かいものとにより分ける道具。「砂を—に掛ける」

————————

篩う（ふるう）

①ふるいに掛けてより分ける。

②選抜する。「試験で—う」

出典：『漢検 漢字辞典（第二版）』

諂う

+ PUSH

諂う（へつらう）

気に入られようと機嫌をとる。こびる。おもねる。「権力者に—う」

————————

諛う（へつらう）

相手に気に入られるよう、言葉で機嫌をとる。言葉でこびる。「上司に—う」「親が子に—う」

出典：『漢検 漢字辞典（第二版）』

備考：『デジタル大辞泉』では「諂う」と「諛う」は一致

————————

倖う（へつらう）

出典：『漢字ペディア』

屠る

+ PUSH

屠る（ほふる）

①鳥獣の体を切り裂く。「羊を—る」

②みなごろしにする。

③敵を打ち破る。「大関を—る」

出典：『漢検 漢字辞典（第二版）』

薨る

+ PUSH

薨る（みまかる）

昔、皇族や三位以上の人が死ぬ。薨去する。「死ぬ」の尊敬語。

「関白は病に臥して—った」

出典：『漢検 漢字辞典（第二版）』

邀える

+ PUSH

邀える（むかえる）

相手の攻撃を待ち受けて、さえぎり止める。

>「魔王のビームを—えて姫様を守った」

出典：『漢検 漢字辞典（第二版）』

咽ぶ

+ PUSH

咽ぶ（むせぶ）

①飲食物や煙などで息がつまりそうになる。

②感情がこみ上げて声がつまる。「あまりの嬉しさに涙に—ぶ」

③風や水の音が激しく泣くように聞こえる。

出典：『漢検 漢字辞典（第二版）』

盲いる

+ PUSH

盲いる（めしいる）

視力を失う。目が見えなくなる。

「お夏の—・いた目の前には」〈白鳥・牛部屋の臭ひ〉

出典：『デジタル大辞泉』

嬾い

+ PUSH

嬾い（ものうい）（懶い）

心がはれやかでない。何をするにもおっくうで、気が進まない。憂鬱で気が晴れない。「—曇り日の昼下がり」

————————

懶い（ものぐさい）

①物事をするのが、なんとなく気が進まない。わずらわしく、おっくうである。「寒くて何をするのも—い」

②体がだるく、気分がすぐれない。

出典：『漢検 漢字辞典（第二版）』

憔れる

+ PUSH

憔れる（やつれる）（悴れる）

（「悴れる」と同義と見做す）

疲れて体がやせ、気力が衰える。「心労が重なって—れる」「入院生活ですっかり—れた」

出典：『漢検 漢字辞典（第二版）』
